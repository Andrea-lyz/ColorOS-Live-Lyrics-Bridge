# ColorOS Live Lyrics Bridge v3.0.1

v3.0.1 是基于 v3.0.0 的性能与渲染稳定性维护版本，重点降低未使用锁屏歌词时对 SystemUI 的后台开销，并继续收紧只应作用于歌词界面的保护逻辑。

## SystemUI 性能优化

- 歌词区域隐藏后立即停止逐帧刷新、可见性恢复与延迟回调；不会因已隐藏的歌词界面继续保留刷新循环。
- Provider 广播在主线程只做轻量校验和快照，歌词解析转入有界后台执行器，降低切歌和媒体状态变化时的主线程压力。
- 歌词局部 View Hook 在无歌词模型、非歌词视图或无活动歌词界面时更早返回，并避免普通路径的临时对象分配与重复祖先查找。
- 移除全局日志 Hook，收紧 OPlus 白名单绕过范围；RecyclerView 通知保护仅作用于官方歌词 Adapter，不再触及普通 SystemUI 列表。

## 渲染与稳定性

- 修复锁屏岛界面切换时偶发回落到官方大字歌词、短暂闪烁后才恢复模块渲染的问题；过渡期间使用有时限的歌词局部恢复机制，不保留后台刷新。
- Apple Music 在同一播放会话切歌时，允许已确认的最新 generation 歌词绕过陈旧的 SystemUI 曲目标题完成晋升，减少歌词已到达但仍显示“暂无歌词”的情况。
- 保持 v3.0.0 已验证的亮屏到 AOD 歌词首帧与布局时序，不引入新的 AOD 可见性、透明度或尺寸门控。

## 日志与安装

- 发布版默认只保留必要警告和错误；详细歌词诊断仍需显式开启对应 debug tag。
- Bridge 内部版本更新为 `3.0.1` / `versionCode 115`。LyricProvider 内部版本未变化，已安装的 Provider 无需重装；Release 仍附带全部 Provider APK 与合集，便于首次安装或补装。
