# ColorOS Live Lyrics Bridge v2.7.2

这一版继续收紧外部 Provider 的切歌接管和歌词来源兜底，重点改善汽水音乐、酷狗音乐/概念版在实际锁屏场景里的稳定性，同时保持 Bridge 渲染链路的发布版日志更安静。

## 新增

- 新增汽水音乐官方运行时歌词对象兜底：本地网络缓存和数据库还没命中时，可直接接管播放器当前歌词对象，降低第二次打开后一直“暂无歌词”的概率。
- 新增汽水音乐数据库表发现和 NetCacheLoader 递归查找能力，兼容不同安装来源或缓存目录结构差异。
- 新增汽水音乐中文翻译语言键兜底，优先识别 `ZH-HANS`、`ZH-CN` 等翻译字段。

## 优化

- 优化汽水 Provider 切歌重试、缺失缓存和 Bridge 发送链路，减少歌词 pending 后长期不落地的情况。
- 优化酷狗 Provider：原版酷狗与概念版酷狗分离处理歌词和封面，概念版不再覆盖系统封面；歌词发送增加同曲目去重和稳定元数据保留。
- 优化 Bridge 对汽水音乐和酷狗概念版的外部歌词接管：短暂 metadata/lyricInfo 抖动时保留稳定歌词表面，降低切换时闪回官方渲染或非歌词区域残留。
- 优化翻译按钮识别和可见性跟随，避免歌词出现后按钮视图被回收或隐藏后不恢复。
- 降低 Bridge、汽水 Provider、酷狗 Provider 的发布版日志噪声，高频诊断默认关闭，仅保留必要异常与显式 debug tag 输出。
- 酷狗 Provider 更新到 `1.1.2` / `versionCode 7`。
- 汽水 Provider 保持 `1.1.0` / `versionCode 2`，安装说明提示在 LSP 管理器中为汽水音乐开启“还原内联钩子”。
